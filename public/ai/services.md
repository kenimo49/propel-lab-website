## 1. LearningMate（教育コンテンツ事業）

### サービス概要
LLM活用に特化した教育コンテンツの企画・制作・販売を行っています。実務で培ったノウハウを体系化し、書籍・オンライン講座として提供します。

### 対象ユーザー
- AIを開発に取り入れたいエンジニア
- Claude Code / GitHub Copilotを実務で使いたい開発者
- LLMOを自社サイトに導入したいマーケター・経営者

### 提供形態
- **Kindle書籍**: 日本語・英語の2言語展開
- **Zenn Books**: 技術書オンライン販売
- **Udemy講座**: 動画コース（準備中）

### 技術領域

## 2. AI-Loop（マーケティング自動化）

### サービス概要
LLMエージェントを活用したSNS運用自動化・コンテンツマーケティング支援サービスです。

### 解決する課題
- SNS運用の工数削減
- データ駆動のコンテンツ戦略立案
- 複数プラットフォームの一元管理

### 技術
- LLMエージェントによる投稿案自動生成
- インプレッション・エンゲージメント分析
- プラットフォーム横断のパフォーマンス比較

---

## 3. AutoCrew（業務自動化コンサルティング）

### サービス概要
LLMによるマルチエージェントシステムの構築・業務プロセスの自動化コンサルティングを提供します。

### 解決する課題
- 定型業務の自動化
- 複数ツール間の連携
- AIエージェントの導入設計

### 技術
- マルチエージェントアーキテクチャ
- MCP (Model Context Protocol)
- CLI Skill設計
- ワークフロー自動化
