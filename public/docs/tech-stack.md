# Propel-Lab — 技術スタック詳細

## コーポレートサイト
- **フレームワーク**: Astro（静的サイトジェネレーター）
- **スタイリング**: Tailwind CSS v4
- **言語**: TypeScript
- **ホスティング**: GitHub Pages
- **LLMO対応**: JSON-LD, llms.txt, URL.mdパターン

## LearningMate（教育コンテンツ）
- **AI**: Claude Code, Context Engineering
- **配信**: Udemy, Zenn
- **制作**: Markdown + AI生成

## AI-Loop（マーケティング自動化）
- **コア**: LLMエージェント
- **分析**: インプレッション分析
- **自動化**: SNS投稿スケジューリング

## AutoCrew（業務自動化）
- **アーキテクチャ**: マルチエージェント構成
- **プロトコル**: MCP（Model Context Protocol）
- **インターフェース**: CLI Skill
- **オーケストレーション**: タスク分解・依存関係管理

## Yureru（音楽・クリエイティブ）

- **画像**: Stable Diffusion
- **デザイン**: Canva
