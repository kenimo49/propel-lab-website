# 合同会社 Propel-Lab — プロジェクト情報

## 進行中のプロジェクト

### LearningMate 書籍シリーズ
- **ステータス**: 継続中
- **概要**: エンジニアのためのAI実践シリーズとして、Claude Code・LLMO・コンテキストエンジニアリングに関する技術書を出版
- **実績**: Kindle 4冊出版済み（日英2言語）、Zenn Books 3冊
- **次の展開**: Udemy講座の準備、新刊の執筆
- **Amazon著者ページ**: https://www.amazon.co.jp/stores/author/B0GQNPRCGF

### LLMO最適化サイト構築
- **ステータス**: 完了・運用中
- **概要**: LLMOに最適化された企業サイトをAstro + GitHub Pagesで構築
- **技術**: Astro, Tailwind CSS, JSON-LD (11スキーマ), llms.txt, URL.mdパターン, Playwright
- **公開リポジトリ**: https://github.com/kenimo49/propel-lab-website
- **成果**: 5段階AI理解構造の実装、33テストによるLLMO品質保証

### Yureru 音楽プロジェクト
- **ステータス**: 継続中
- **概要**: AI技術を活用した音楽制作・配信プロジェクト
- **実績**: 409本以上の楽曲を公開、YouTube登録者3,815人
- **アーティスト**: Yumei（メインボーカル）、Silk Reign（洋楽バラード）
- **配信**: YouTube, Spotify, Apple Music

### Persona Platform（研究開発）
- **ステータス**: 開発中
- **概要**: 性格データ（MBTI/エニアグラム/DiSC/ビッグファイブ等）をハブとした、パーソナライズドサービス基盤
- **技術**: FastAPI, React, Docker
- **ビジョン**: 性格データを共有することで、すべてのアプリが「自分専用」になるプラットフォーム
